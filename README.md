## Getting Started

### Prerequisites

Make sure you have the following installed:

- Node.js and npm
- Vue CLI

### Installing Dependencies

```bash
npm install
```

### Compiles and hot-reloads for development

```bash
npm run serve
```

### Lints and fixes files

```bash
npm run lint
```
